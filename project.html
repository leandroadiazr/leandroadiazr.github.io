SoftDevJobs

SoftDevJobs Project is an App that allows Software Developers and also the general public to find Jobs based on the USA.GOV platform.

You will find on this first release version features that will help you stay organized and make your job search easy.

Features:

  • Latest Jobs for Developers and General Public.
  • Favorites Tab where you can add / organize / remove jobs that that match your interests.

    RehabCalculator
