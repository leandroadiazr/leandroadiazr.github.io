SoftDevJobs

SoftDevJobs Project is an App that allows Software Developers and also the general public to find Jobs based on the USA.GOV platform.

You will find on this first release version features that will help you stay organized and make your job search easy.

Features:

  • Latest Jobs for Developers and General Public.
  • Favorites Tab where you can add / organize / remove jobs that that match your interests.
  • Light / Dark Mode Adaptable.

    • Requirements:

  • Must have iOS 13.2 or later installed.

    • Environments:

  • Only Available on iPhone.

    • About This App

    The original idea was to make an app that only software developers looking for jobs will have access to, but in the building phase I realized how much potential this project has, Specially when looking for goverment jobs.

    This app was build in it's entirely, programatically using only Swift UIKit Framework, no external Libraries
